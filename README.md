# PerfilApp - Hexagonal Architecture with DDD

1. Clonar proyecto
2. Instalar dependencias
```
npm i
```
3. Clonar el archivo
 ```
 .env.template
 ```
4. y renonmbrarlo a 
```
.env
```
5. Cambiar las variables de entorno
6. Levantar base de datos
```
docker-compose up -d
```
